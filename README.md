# 核心特色

# 全能Agent - 一个Agent处理所有职业发展需求
# 智能识别 - 自动理解用户意图，无需选择功能
# 自然对话 - 像与人交谈一样简单直接
# 全面覆盖 - 职业规划、简历、面试、技能、薪资等所有方面
